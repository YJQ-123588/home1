# 西游记 - 祸起观音院

一款基于Pygame开发的2D RPG冒险游戏，以中国古典名著《西游记》中"祸起观音院"为故事背景。

## 游戏特点

- 经典的2D RPG游戏玩法
- 回合制战斗系统（支持普攻、技能、大招、避战）
- 多场景探索（村庄、寺庙）
- 丰富的NPC对话和剧情
- 精美的像素风格角色动画（支持多状态动画）
- 背景音乐和音效支持（战斗音效分级）
- 存档/读档系统（最多3个存档位，支持存档/覆盖/删档）
- 游戏界面状态栏显示角色属性和技能信息
- 右上角小地图显示玩家、怪物和NPC位置
- 场景渐入渐出过渡效果

## 游戏操作

### 移动控制
- **方向键/WASD**：控制孙悟空移动
- **Shift**：加速移动
- **空格/回车**：关闭对话框、确认选择
- **ESC**：打开菜单（继续游戏/返回村庄/读取存档/游戏介绍/退出游戏）

### 战斗操作
- **数字键1**：普通攻击（伤害=玩家攻击力）
- **数字键2**：技能攻击-如意金箍棒（伤害=玩家攻击力+35）
- **数字键3**：大招-七十二变（伤害=玩家攻击力+75，需长老2解锁）
- **数字键4**：避战（扣除20HP退出战斗，不击败怪物）
- **鼠标点击按钮**：也可选择攻击方式

### 存档操作
- **ESC键**：打开菜单，可选择存档/读档
- **读档界面**：点击空槽位可存档，点击"进入游戏"读取存档
- **数字键1/2/3**：有存档时读取，空槽位时保存
- **删档**：选中存档后按Delete键或点击删除按钮（有确认对话框）

## 游戏流程

1. **开始界面**：点击"开始游戏"按钮开始，也可读取存档或查看游戏介绍
2. **村庄场景**：与NPC对话，了解剧情，接受增益（长老1增加HP上限、长老2解锁大招、长老3治疗）
3. **前往寺庙**：与土地公对话后，通过村庄出口前往寺庙（有确认对话框）
4. **寺庙场景**：探索寺庙，遭遇小怪（M1-M3），可选择战斗或避战
5. **BOSS出现**：击败所有小怪后，牛魔王（M4）才会出现
6. **战斗系统**：回合制战斗，选择攻击方式击败敌人
7. **战败救援**：战斗失败后土地公将玩家救回村庄
8. **游戏通关**：成功击败牛魔王，降服观音院妖怪

## 运行要求

### Python版本
- Python 3.6+

### 依赖库
- Pygame 2.0+
- pytmx（用于加载Tiled地图）

### 安装依赖
```bash
pip install pygame pytmx
```

## 安装与运行

1. 克隆或下载本项目
2. 确保已安装Python 3.6+
3. 安装依赖库：
   ```bash
   pip install pygame pytmx
   ```
4. 运行游戏：
   ```bash
   python main.py
   ```

**注意**：游戏资源文件已包含在 `resources/` 目录中，无需额外下载。

## 游戏配置

游戏配置文件位于 `config/settings.py`，可调整以下参数：

- **窗口设置**：屏幕尺寸（800x600）、帧率（30FPS）
- **玩家属性**：移动速度（6）、基础生命值（100）、基础攻击力（25）
- **战斗设置**：普通攻击伤害（=玩家攻击力）、技能伤害（=攻击力+35）、大招伤害（=攻击力+75）
- **怪物属性**：普通怪物生命值（100）、攻击力（15）、BOSS生命值（250）、攻击力（30）
- **NPC设置**：移动速度（1）、徘徊步数（60）
- **小地图**：宽度（180）、高度（140）、标记大小（玩家6、怪物5、BOSS6、NPC5）
